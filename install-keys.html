SiteOS

System Setup

To finish setting up your SiteOS instance, follow the instructions below:

- Update your config.ini file (located at /home/config.ini). Add your Mailgun private key, your AWS access keys, and your AWS S3 settings.

- Set up inbound email for Mailgun as shown in this link, and add the following route to store inbound emails "https://app.[your domain name].com/mail/email-store.php".

- Set up your AWS credentials file.

- Run caddy restart to restart the web server. Your SiteOS instance should then be accessible by going to https://app.[your domain name].com
